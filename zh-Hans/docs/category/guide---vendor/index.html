指南 - 供应商

学习如何编写供应商接入外部系统

📄️ 什么是供应商

供应商 (Vendor) 是 Yuan 中的一种拓展程序 (Extension)，它连接 Yuan 和外部系统，充当了 Yuan 和外部系统之间的桥梁。

📄️ 跨语言互操作指南

如果提供商 SDK 的语言不是 JavaScript 或者通用的 HTTP 等应用层协议，情况会变得比较复杂。

📄️ 提供账户信息

对接交易所时，供应商需要将在交易所开立的账户，转换成 Yuan 的标准账户信息。

📄️ 提供历史行情数据

行情数据有若干种不同的具体类型，但都可以归结为对特定类型，在特定的时间范围内的，对特定序列的查询。

📄️ 提供历史订单数据

订单 可以归结为对特定账户，在特定的时间范围内的查询，也是一个序列采集的问题。与历史行情数据类似，供应商需要提供历史订单数据。

📄️ 提供产品规格

提供商有责任提供产品规格信息。

📄️ 提供实时行情数据

供应商可以通过频道提供实时行情数据，以便于其他终端订阅。

📄️ 提供交易接口

在 Yuan 中，交易接口是指用于与外部系统进行交易操作的接口。所有的交易接口都围绕着 订单 这一概念进行。

📄️ 提供转账接口

您可能需要阅读 基础概念 / 转账 来深入理解 Yuan 中转账的基本概念。
